# Complete_backend

## Node.js + Express Setup Notes

### 1) Initialize Node.js project
Run:
`npm init -y`

This creates a default `package.json` file.

### 2) Install Express
Run:
`npm i express`

This installs Express inside the `node_modules` folder.

### 3) Next step
Now you can create and run an Express server using `server.js`.
